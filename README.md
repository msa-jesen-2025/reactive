# Naloga - Reaktiven zaledni sistem Quarkus

V tem repozitoriju se nahaja reaktiven Quarkus zaledni sistem - Measurements.

Vaša naloga je, da na reaktiven način implementirate metodo ``putProduct`` v razredu [ProductController.java](backend_reactive_naloga/src/main/java/si/um/feri/measurements/rest/ProductController.java). V komentarju je zapisana klasična implementacije te metode, ki vam je lahko v pomoč.
